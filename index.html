MY PROJECTS

MANGROVISION
Mobile Application

MangroVision is an interactive and informative mobile application reinforced machine learning model intended for learning and identifying ten (10) local mangrove species, developed to serve as a modern and less cumbersome method of mangrove species identification. It is a system where users can recognize a mangrove by giving leaf image as an input, the output will be an image and details of scientific name and the level of accuracy of the identified leaf. It also provides significant information about mangrove and its ecosystem.

SAFE ZONE
Website Application

SAFE ZONE (SZ) is a website application implemented by The Sentinels, a non-profit organization, that aims to reclaim and defend the cyberspace of women and discriminated gender against ICT-GBV in the Philippines. It is a report filing system that uses AI and text labelling to automatically predicts what kind of ICT facilitatd gender-based violence the user experienced. The system can also give tips to the user on how to cope up with the situation and what are the possible liabilities of the offender.

ASU EVENT SEARCH APP
Mobile Application

A multi-platform software that enables students to search events and activities exclusive for all campuses of Aklan State University (ASU). Faculties of ASU can post events that aims to strenghten the community of learners and teachers. Students can bookmark events and receive notifications from the app. The goal of this software is to provide a platform to rapidly dissimiante event details to all students and teaching staffs of ASU.
